# AibbyScope
ロームのハードウェアFFT内蔵マイコン ML63Q2557を使ったオーディオ帯スペアナ

# 概要
ROHM ML63Q2557マイコンに内蔵されるハードウェアFFTエンジンを使用したスペクトラム・アナライザのデモンストレーションプログラムです。
オシロスコープを表示機として使うことで、最少の部品を使って実験室で手軽に試すことができます。  

[![Youtube](https://github.com/user-attachments/assets/3bf2b3cd-ec55-4929-a7e0-7f8a2fe3e476)](https://youtu.be/u3YaAkrQtn4)

# 制限事項
* サンプリング周波数はクロックに使う水晶振動子の精度の影響を受けます。
* サンプリング周波数は1Hz刻みで変更できますが、クロックを分周してタイミングを作っているため、厳密な正確さはありません。
* FFT表示の一番左はDCオフセット値、その後各周波数ごとのレベルが表示されます。
* デモンストレーションプログラムです。精密測定・業務などへの利用には適しません。

# 使い方
* アナログ入力端子A4(P36)にアナログ電圧を入力します。
* P40～P47に抵抗で作ったR-2R DACを接続し、出力をオシロスコープへ入力します。
* P52,P53にUARTでPCに接続し、ターミナルソフト経由で操作します。通信設定は115200bps 8N1です。

# ターミナル上のキーアサイン
| KEY | 役割 |
|---|---|
| w | 波形表示|
| f | FFT表示 |
| ( r ) | サンプリングレート入力 または 現在の設定÷2、×2 |
| < s > | 表示サンプルサイズ入力 または 現在の設定÷2、×2 |
| - q + | FFT入力波形のゲイン入力 または 現在の設定－1、＋1 |
| n h | FFT前の窓関数 n:矩形窓 h:ハニング窓 |
| t | VSYNCタイミング調整値入力 |
| d | ドットクロック タイミング調整値入力 |
| ? | ヘルプ表示 |
| SPACE | サンプリング一時停止・再開 |
| ESC | デフォルト設定に戻す |

# 動作仕様
* サンプリング周波数：250～50000Hz 1Hz刻みで自由に設定
* ADC：マイコン内蔵 12bit SAR ADC
* アンチエイリアスフィルタ：アナログ入力回路に依存
* FFT処理：2～1024サンプル入力、2～513点出力、リニア表示
* 表示出力：8bit パラレルR-2Rラダー抵抗DAC出力
* 表示間隔：リフレッシュレート 50Hz固定、掃引：5ms(FFT表示)、10ms(波形表示)

# ハードウェア情報
### ターゲットマイコン
  ROHM ML63Q2557

### ターゲットボード
特に指定なし  
開発は自作ボードAIBBY(アイビー)を使用、欲しい方はご連絡ください

### 最低限必要な周辺部品
水晶振動子 32768Hz  
水晶発振回路用コンデンサ×2  
ディスクリートR-2RラダーDAC(1kΩ×7、2kΩ×9)  
アナログ入力用のVcc/2・プリアンプ回路  
USB-UART変換モジュール

### 接続端子
| 端子 | 役割 |
|---|---|
| P36 | アナログ入力A4、Vref=Vcc |
| P40～47 | パラレルDAC出力 |
| P70 | 画像表示開始トリガー |
| P72 | 動作確認用LED |
| P53 | UART TX出力 115200,8n1 |
| P52 | UART RX入力 115200,8n1 |
| P21 | デバッグ用出力、処理時間計測信号 |

### 開発環境
ROHM LEXIDE-Ω、WCH-LinkEデバッガ
